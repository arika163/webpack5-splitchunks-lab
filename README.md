# webpack5-basic

A minimal Webpack 5 demo project (no framework) that shows:

- usage of common third-party libs: `lodash`, `axios`, `dayjs`
- dynamic import to create a separate chunk
- `optimization.splitChunks` to extract `vendors`

Quick start:

```bash
cd webpack5-basic
npm install
npm run build
```

Development:

```bash
cd webpack5-basic
npm start
```
